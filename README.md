Construir
=========

Usar sudo o permisos de root

```bash
$ sudo lb config
$ sudo lb build
```

En caso de tener un apt-cacher corriendo se puede hacer:

```bash
$ sudo CACHER=127.0.0.1:3142 lb config
$ sudo CACHER=127.0.0.1:3142 lb build
```
